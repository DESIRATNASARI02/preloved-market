# preloved-market
Website preloved item yang menyediakan berbagai barang dengan kualitas menengah hingga hampir baru. Daripada tidak digunakan lagi, website ini hadir sebagai solusi bagi pengguna yang mencari produk berkualitas dengan harga terjangkau, serta bagi penjual yang ingin memaksimalkan kegunaan item yang mereka miliki.
